## Usage

Install the theme:

 ```bash
git clone https://github.com/jhia/hexo-theme-pinktage.git themes/pinktage
```


Install the dependencies:

```bash
npm i -S hexo-renderer-sass hexo-renderer-markdown-it markdown-it-emoji twemoji
```


Use the theme:

```yml
# config.yml
# theme: landscape
theme: pinktage
```


Preivew:

```bash
hexo server
```
